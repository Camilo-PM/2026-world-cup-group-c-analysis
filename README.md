2026 FIFA World Cup - Group C Analysis

Data-driven analysis of the teams competing in Group C of the FIFA World Cup 2026.

Teams Analyzed
Brazil
Morocco
Scotland
Haiti
Project Objective

This project evaluates the recent performance of each national team using match data from their latest 10 games, including official matches and friendlies.

The analysis focuses on:

Recent form
Offensive efficiency
Defensive stability
Goal difference
Performance trends
Comparative power rankings

Main Findings
Morocco achieved the highest Power Score based on recent performances.
Brazil remains one of the strongest offensive teams in the group.
Scotland showed competitive form but weaker defensive metrics.
Haiti appears as the group's underdog while still maintaining attacking potential.
